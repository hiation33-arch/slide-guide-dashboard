# 해보는 언니 — 슬라이드 가이드 대시보드

강의 슬라이드 제작 시 용도·세트·팔레트를 선택하면 Claude용 슬라이드 제작 가이드가 한 번에 복사되는 도구입니다.
파일을 서버로 보내지 않습니다 — 모든 동작은 브라우저 안에서만 처리돼요.

## 바로 사용하기

👉 [해보는 언니 슬라이드 대시보드 열기](https://hiation33-arch.github.io/slide-guide-dashboard/)

---

## 주요 기능

- **용도 선택** — 강의 슬라이드 / 공모전 제출 / 제안서·사업계획서 중 선택
- **슬라이드 세트 선택** — 미니멀 라인 / 카드+아이콘 / 매거진 / 다이나믹 / 기하학 5가지
- **컬러 팔레트 선택** — 머스터드 / 소프트 틸 / 포레스트 그린 / 더스티 블루 / 차콜 5가지
- **한 번에 복사** — 용도별 슬라이드 뼈대 + 세트 스타일 + 컬러 가이드가 한 번에 클립보드에 복사
- **로컬 처리** — 가입·로그인·서버 없음. 브라우저만 있으면 바로 동작

---

## 사용법

1. **용도 선택** — 강의 슬라이드 / 공모전 / 제안서 중 제작 목적에 맞는 용도를 클릭합니다.
2. **세트 선택** — 5가지 슬라이드 세트 중 원하는 스타일을 클릭합니다. 썸네일로 미리 확인할 수 있어요.
3. **팔레트 선택** — 강의 성격에 맞는 컬러 무드를 선택합니다.
4. **복사** — 상단 "용도 + 세트 + 팔레트 복사" 버튼을 누르면 Claude용 가이드가 클립보드에 복사됩니다.
5. **Claude에 붙여넣기** — Claude 채팅창에 붙여넣고, 노트북LM에서 가져온 강의 내용을 추가하면 슬라이드가 완성됩니다.

## 용도별 슬라이드 뼈대

**강의 슬라이드 (15~25장)**
Cover → 오프닝 질문 → 오늘의 목표 → 배경 설명 → 핵심 개념 → 사례/시연 → 참여 활동 → 핵심 정리 → Q&A/Closing

**공모전 (5~7장)**
Cover → 문제 정의 → 아이디어 소개 → 실행 방식 → 기대 효과 → 차별성/경쟁력 → Closing

**제안서·사업계획서 (10~15장)**
Cover → Executive Summary → 문제 정의 → 시장·환경 분석 → 제안 솔루션 → 실행 계획 → 조직·운영 체계 → 예산·자원 계획 → 기대 효과 → Closing

## 워크플로우

```
노트북LM (강의 내용 → 보고서/문서 형식 출력)
        ↓
대시보드에서 용도 + 세트 + 팔레트 선택 → 복사
        ↓
Claude 채팅창에 붙여넣기 + 노트북LM 출력 내용 추가
        ↓
Claude가 HTML 슬라이드 생성
```
