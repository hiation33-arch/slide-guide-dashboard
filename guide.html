USAGE GUIDE
해보는 언니 슬라이드 대시보드
사용 가이드
대시보드에서 선택만 하면 Claude가 바로 슬라이드를 만들어드려요.
처음이라도 3분이면 충분해요.
BASIC WORKFLOW
1
NotebookLM
강의 내용 준비
NotebookLM에서 강의 주제를 정리하고 보고서/문서 형식으로 출력해요.
이렇게 활용해요
강의 소스 자료를 NotebookLM에 업로드
학습 가이드 또는 문서 형식으로 출력 요청
출력된 내용을 복사해서 메모장에 임시 보관
2
대시보드
용도 · 세트 · 팔레트 선택
대시보드에서 세 가지를 순서대로 선택하면 복사 버튼이 활성화돼요.
①
용도 선택
강의자료 / 공모전 / 제안서
②
세트 선택
A~E 중 스타일 선택
③
팔레트 선택
무드에 맞는 컬러
3
대시보드
프롬프트 복사
우측 상단의 용도 + 세트 + 팔레트 복사 버튼을 클릭하면 슬라이드 제작 가이드가 클립보드에 복사돼요.
4
Claude
Claude에 붙여넣기
복사한 가이드 + NotebookLM에서 정리한 내용을 함께 붙여넣으면 Claude가 HTML 슬라이드를 생성해요.
붙여넣기 순서
복사한 슬라이드 가이드 먼저 붙여넣기
NotebookLM에서 정리한 강의 내용 추가
추가 요청사항 있으면 마지막에 덧붙이기
5
완성
HTML 슬라이드 완성
Claude가 브랜드 시스템이 적용된 HTML 슬라이드 파일을 생성해요. 다운로드 후 Chrome에서 바로 열어서 사용하거나 수정 요청을 추가로 드릴 수 있어요.
